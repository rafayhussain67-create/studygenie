Solve: 2x² + 5x - 3 = 0

Solving using the Quadratic Formula

x = (-b ± √(b² - 4ac)) / 2a
Step 1 — Identify valuesa = 2, b = 5, c = -3
Step 2 — Calculate discriminantb² - 4ac = 25 - 4(2)(-3) = 25 + 24 = 49
Step 3 — Apply formulax = (-5 ± √49) / 4 = (-5 ± 7) / 4
Step 4 — Final answersx = 0.5 or x = -3 ✓
